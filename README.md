# Rave Contracts

### Install
```sh
git clone https://github.com/rave-names/contracts
cd contracts
npm install -D
```

### Lint
```sh
npm run lint
```

### Prettify
```sh
npm run prettier
```

### Compile
```sh
npm run compile
```

### Verify
```sh
npx truffle run verify CONTRACT_NAME@CONTRACT_ADDRESS
```
